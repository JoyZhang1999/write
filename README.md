
简介
AI Air Canvas - Silk Smooth 是一个基于 MediaPipe Hands 模型的创新型空中书写应用。它允许用户通过实时手势与虚拟画布进行交互，实现隔空书写、工具切换、缩放和平滑的粒子消散效果。

本版本特别采用了镜像书写模式，用户在书写时会感受到字迹是左右反转的，模拟了透过屏幕或玻璃看字的独特体验，同时确保所有交互光标都与用户在镜像画面中的视觉保持同步。

核心特性
📐 镜像书写：手部向右移动时，笔迹向左延伸，提供独特的“隔屏书写”体验。

✨ 丝滑笔迹：使用二次贝塞尔曲线算法平滑处理手势点，消除抖动，使空中书写效果如同流畅的墨迹。

🖐️ 智能 UI 交互：通过张开手掌手势快速弹出/隐藏半透明磨砂玻璃 UI 面板，并使用食指尖悬停触发选择功能。

🔎 双指缩放：双手伸出食指，靠近或远离即可缩放和移动画布。

❄️ 粒子消散特效：通过特定的“剪刀手”手势，将画布上的所有笔迹转化为粒子，以雪花般消散。

🛠️ 完整的工具集：提供多种颜色、大小和笔/橡皮擦工具供用户选择。

🛠️ 技术要求与使用指南
本项目是一个单文件 HTML 应用，无需复杂的安装配置，仅依赖浏览器和网络连接。

1. 技术栈
HTML/CSS/JavaScript

MediaPipe Hands v0.9 (用于高精度手势识别)

Canvas API (用于高性能绘图和特效)

2. 如何运行
将完整的代码保存为 index.html 文件。

使用 Chrome 或其他支持 MediaPipe 的现代浏览器打开此文件。

首次运行时，浏览器会请求摄像头权限，请务必允许。

手势图标,手势名称,动作描述,功能
🖐️,张开手掌 (Palm Open),在摄像头前完全张开手掌，维持 1 秒。,切换 UI 菜单的显示/隐藏（暂停模式）。
👌,捏合 (Pinch),食指和拇指尖轻轻接触或极度靠近。,进入 书写/橡皮擦 模式。
✌️,剪刀手 (Peace),食指和中指伸出，其他手指弯曲。,清空画布并触发粒子消散特效。
↔️,双指伸出 (Two Index),两只手同时伸出食指。,缩放画布：靠近 (缩小) 或远离 (放大)。
👆,食指指向 (Point),在 UI 模式下，用食指尖指向 UI 元素。,选择工具：悬停 0.8 秒触发按钮选择。
